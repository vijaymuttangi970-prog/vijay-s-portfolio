## 🖥️ About

This is a single-page, responsive portfolio built with plain HTML & CSS. It includes:

- **Hero section** — intro and quick call-to-action buttons
- **About** — education, year, languages known, and contact info
- **Skills** — C, Python, and Problem Solving with progress indicators
- **Projects** — Weather Prediction System (Python) and 2D Graphics Editor (C)
- **Contact** — phone, email, and GitHub links

## 🛠️ Tech Stack

- HTML5
- CSS3 (custom properties, gradients, animations)
- Vanilla JavaScript (scroll-based fade-in animations, active nav highlighting)

## 🚀 Getting Started

To run locally:

1. Clone the repository
   ```bash
   git clone https://github.com/vijaymuttangi970-prog/<your-repo-name>.git
   ```
2. Open `index.html` in your browser — no build steps or dependencies required.

## 🌐 Deploying on GitHub Pages

1. Push this repo to GitHub.
2. Go to **Settings → Pages**.
3. Under "Source", select the `main` branch and `/root` folder.
4. Save — your site will be live at `https://vijaymuttangi970-prog.github.io/<your-repo-name>/`.
